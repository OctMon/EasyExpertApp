# 自动打包输入更新日志上传发布应用到Fir,Pgyer

## Requirements
Xcode 9.0+

## Installation
复制expert.sh到项目根目录

## Example
```shell
$ sh expert.sh
输入序号,选择打包方式,按回车继续 
1. development 
2. ad-hoc      
3. app-store   
4. enterprise  
```
